## Installation

### Using create-rapin-app

With help npx:

```bash
yarn create rapin-app <project-name>
```

or with yarn

```bash
npx create-rapin-app <project-name>
```

To run in developer mode use the command:
```bash
yarn run dev
```

To run in production use the command:
```bash
yarn start
```
